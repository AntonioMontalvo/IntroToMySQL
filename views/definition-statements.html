Database

We can define a database as a collection of organized information. This information is stored in different platforms, anywhere from someone's personal computer to dedicated storage facilities offered by Amazon, Google etc..

RDBMS - relational database management system

A relational database is a management system that stores data in the form of related tables where, a single database can be spread across serveral tables. It makes easier to see how data is related and extracted from the database.

How to create a Database

MySQL uses commands that are recognizable and easy to remember. Let's create a database and do stuff with it. Run this command: CREATE DATABASE playground_1_db;

Viewing all databases

Once you've created a database you can use the following command for MySQL to show all databases available. SHOW DATABASES;

Deleting a database

Imagine you no longer want to have a particular database. You can just dropt it: DROP DATABASE playfround_1_db; Go ahead and create the same database again after you drop it.

Accessing a database

Once a database has been created, you have to point to it so you can access it.

USE playground_1_db;

Creating tables

Ok, we finally have a database named and we can use it. Now in order to introduce data we have to name and create the table/s. When creating tables is paramount to structure it sensibly so later on we can use it with ease.

Data Types

When we store data we have to remember that we are using X amount of bytes. Each Data type has its own storage required, for example Dates of Type YEAR is 1 byte where as DATETIME is 5 bytes + fractional seconds storage. The point here is to always keep in mind how many bytes you use to store your data. See all different Data Types below

  • Numeric Type
  • Date and Time Types
  • String Types
  • JSON Type
  • Spacial Type

Always define Data Types according to the values you want to store and keeping in mind memory allocation.

Primary Key

When storing data we should ad an id column so we can identify each row of the table . We can use auto increment along to primary key.

Quering all tables

Use this query to list all tables in the database.

SHOW TABLES;

See the structure of an existing table.

EXPLAIN users;

Deleting tables

Try it if you want to with:

DROP TABLE users;

Modifying tables

Add to the table

Let's say that you forgot to add a row or that in the future you want to add other data to your table. In MySQL this is known as ALTER TABLE Since this is an introduction we will use only a basic example. Lets add an email and state row for each user.

Remove from table

Use this query for testing if you want. Be cautious, this will permanently remove the data.

ALTER TABLE users DROP email;

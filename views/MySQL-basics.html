What is MySQL?

MySQL is an open source relational database management system that runs as a server providing multi-user access to a number of databases. It is named after co-founder Michael Widenius' daughter, My. The SQL phrase stands for Structured Query Language. The MySQL development project has made its source code available under the terms of the GNU General Public License, as well as under a variety of proprietary agreements. MySQL was owned and sponsored by a single for-profit firm, the Swedish company MySQL AB, now owned by Oracle Corporation.

How can I install it?

Go to this page MySQL Community Downloads and download MySQL Community Server and MySQL Workbench where MySQL Community Server is the world's most popular open source database and MySQL Workbench is a visual database design application to manage and document database schemata. For Windows Operating System, once you have downloaded the Community Server, upon installation select MySQL Command Line Client if you wish to work from Bash. If you don't wish to use the command line for your project you'll be just fine with Workbench. However in this tutorial we will be using terminal with our queries.

Windows Operating System can use Wamp Server

The Wamp Server program and environment allows you to create web applications with PHP Apache2 and MySQL database. Wamp Server Download. As everything Windows, things can always get a bit messy but this tutorial is not about installation, is an introduction to MySQL.

Starting MySQL Server

On Mac OS X

Open System Preferences and click MySQL. This will give you acces to Start MySQL Server.

On Windows

Open from the Command Line Client - Unicode application. This will automatically open the command line window for you.

Connecting to MySQL via Terminal on Mac OS X

Open your Terminal use this path cd /usr/local/mysql/bin to locate mysql. Then use this to access mysql ./mysql -u root -p and enter your password. You should see something as below.
